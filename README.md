South Africa Unemployment Dashboard

Overview
This project is a Power BI dashboard analyzing unemployment trends in South Africa using QLFS data. It provides insights into unemployment patterns across provinces, age groups, gender, and education levels.

Key Insights
- Higher unemployment is observed among individuals with lower education levels.
- The 15-24 age group has the highest unemployment levels.
- Females show a higher unemployment rate compared to males.
- Certain provinces such as Eastern Cape and Gauteng have higher unemployment rates.
